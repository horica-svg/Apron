# Apron - Your Smart Meal Planner

Apron is a mobile application developed with Flutter, designed to help users discover recipes, manage their dietary preferences, and organize their shopping. The app provides a seamless experience for browsing meal categories, filtering recipes based on specific needs (gluten-free, vegan, etc.), and saving favorite recipes.

## Getting Started

With Firebase integration, users can create accounts and have their data synchronized
across sessions.

## Main Features (for now)

- **User Authentication**: Secure registration and login using Firebase Authentication (Email & Password).
- **Recipe Discovery**: Browse through a comprehensive list of recipes, organized by category.
- **Advanced Filtering**: Filter recipes based on dietary restrictions such as gluten-free, lactose-free, vegan, and vegetarian.
- **Favorites**: Mark recipes as favorites for quick and easy access.
- **Shopping List**: Manage a shopping list for necessary ingredients.
